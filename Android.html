Tutorial Membuat Aplikasi Android
Membuat Aplikasi Android Sederhana dengan Java
Tutorial langkah demi langkah membuat aplikasi Android "Hello World" menggunakan Java di Android Studio.
Langkah 1: Buka Android Studio dan buat project baru.
Langkah 2: Pilih Empty Activity, beri nama project, pilih bahasa Java.
Langkah 3: Pada activity_main.xml, ganti layout menjadi: 
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:gravity="center"
    android:orientation="vertical">

    <TextView
        android:id="@+id/text_hello"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello World!"
        android:textSize="24sp"
        android:textColor="#1a237e" />
</LinearLayout>
Langkah 4: Pada MainActivity.java: 
package com.example.helloworld;

import android.os.Bundle;
import androidx.appcompat.app.AppCompatActivity;

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
    }
}
Langkah 5: Jalankan aplikasi di emulator atau perangkat Android.
Hasil: Tampil tulisan "Hello World!" di tengah layar.

Membuat Aplikasi Android Sederhana dengan Kotlin
Tutorial membuat aplikasi Android "Hello World" menggunakan Kotlin di Android Studio.
Langkah 1: Buka Android Studio dan buat project baru.
Langkah 2: Pilih Empty Activity, beri nama project, pilih bahasa Kotlin.
Langkah 3: Pada activity_main.xml, gunakan layout berikut: 
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:gravity="center"
    android:orientation="vertical">

    <TextView
        android:id="@+id/text_hello"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello World!"
        android:textSize="24sp"
        android:textColor="#1a237e" />
</LinearLayout>
Langkah 4: Pada MainActivity.kt: 
package com.example.helloworld

import androidx.appcompat.app.AppCompatActivity
import android.os.Bundle

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
    }
}
Langkah 5: Jalankan aplikasi di emulator atau perangkat Android.
Hasil: Tampil tulisan "Hello World!" di tengah layar.
